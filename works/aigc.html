选集 02

品牌 IP 工作流

小蓝猫 XIAOLANMAO：全链路双引擎（即梦 / ChatGPT）+ 外形闸门与驳回回流；分场景试稿、Prompt 锁定与周边 Mockup 落地。

全链路

小蓝猫 IP · 双引擎并行 + 驳回回流

需求表 → 三视图 + Prompt 模板库 → 即梦批量 / ChatGPT 备选 → 外形 OK？→ PS 精修 → 品牌审稿 → 归档 → 社媒 / 提案输出。

  1. 即梦与 ChatGPT 分支收敛后，须经外形闸门与品牌审稿。
  2. 终稿入库后再进入展会邀请、社媒包等对外渠道。
流程 A

品牌 IP 场景 · 风格参考快速试稿流

三视图 + 场景参考 → 即梦出图 → PS 审稿。试 seed 时外形不得漂移，屏显彩条与胸前 Logo 为硬约束。

  1. 参考图锁形，随机 seed 不改变彩条屏脸与 Logo 位置。
  2. 与 场景营销审稿流 共用 Prompt 模板与负向词库。
场景营销

小蓝猫 IP 场景营销 · AIGC 审稿流

参考图锁形 · 即梦 / ChatGPT 草图 · PS 终稿。品牌色 #0066FF 与三视图不变，模块化 Prompt 批量试生成。

  1. 归档参考图与负向词，固定 seed / 版本号命名。
  2. PS 终稿对齐品牌手册后再进入提案或自媒体。
外形锁定

XIAOLANMAO Prompt 模板 · 外形锁定流

三视图 + 元素拆解 → 可控出图。将五官、屏显彩条、胸前 LpDISPLAY 标识位置写入模板，防止生成跑偏。

  1. 负向词：禁止改耳形、屏脸比例、Logo 位置。
  2. 固定 seed 记录、版本号命名、审稿对比表。
周边示意

IP 周边示意 · 草图 → Mockup 流

帆布袋 · 钥匙扣 · 展会赠品。印刷图必须用定稿 IP，Mockup 只换载体不换角色造型。

  1. 袋印尺寸留白、钥匙扣双面、终稿过审后再打样。
  2. 实拍逐步替换 mockup 进入展会物料库。

IP 定稿与手册见 IP 设计 · XIAOLANMAO 设定。

选集 03

电商视觉工作流

橄榄绿风扇：产品素材 / 版式详情 / 上线迭代三泳道并行，数据回流驱动第二轮优化；模块级合成不重模。

三泳道

橄榄绿风扇 · 电商全链路泳道图

泳道 A 产品素材（SKU → 渲染库 → 抠图 QC → 主图组）· 泳道 B 版式与详情（网格 → A+ → mockup → 运营审核 → 多语言）· 泳道 C 上线迭代（压缩 → 上传 → AB 测 → 数据回流）。

流程 B

橄榄绿风扇 · 姿态控制 · 详情合成流

渲染源 + 侧面结构参考 → 版式 → 合成。沿用现有渲染角度作结构参考，只合成背景与文案层，风扇本体不重模。

  1. 与 详情长图导出流、背景合成流 衔接使用。
主图合成

橄榄绿露营风扇 · 背景合成流

透明产品渲染 + 场景底图 + 卖点文案。风扇结构、比例与配色不改，仅替换背景与版式模板。

  1. 先定尺寸网格（960×600 主图），再套场景。
  2. A+ 六宫格卖点、移动端竖版裁切并行规范。
详情长图

详情页长图 · 多模块尺寸导出流

笔记本 + 手机双端预览 · 模块切片。先完成详情 mockup 总览，再按主图、卖点、参数表分模块导出。

  1. Figma / PS 切片 → 压缩与 sRGB → 运营后台上传。
  2. 主图、A+、手机竖图三套命名规则并行。
A+ 批处理

跨境 A+ 模块 · 批处理对齐流

主图组 + 功能拆解 + 场景图。以现有渲染为源批量生成统一光感 A+ 模块，不重新建模。

  1. 模块标题与图标层在 PS 统一，产品层仅抠图与阴影。
  2. 动作批处理、对齐安全区、webp / jpg 双格式输出。

对应主图与 A+ 成片见 产品视觉 · 跨境电商。

选集 05

上线质检 · 三分支

渲染 / 版式 / 文案并行 QC → 风险判断 → 上线包或返工；含驳回回流版式与复审合并路径。

质检闸门

上线质检 · 驳回回流版式

渲染 QC、版式 QC、文案 QC 并行检查后合并；通过则打包上线，驳回则返工并回流版式环节复审。

  1. 与 星型审稿中枢 衔接：PS 终稿后再进入上线质检闸门。
  2. 返工虚线回流版式 QC，复审虚线回合并节点。
选集 06

合规要点

生成内容仅作内部提案与草图，对外发布物必须经过品牌审稿与上线质检；电商参数与认证标识以设计稿 + 产品确认为准。

  • IP 五官、比例、主色 #0066FF 不可擅自改动；衍生形象需对照应用手册。
  • 归档试生成记录、Prompt 版本与负向词，便于复盘与交接。
  • 产品渲染结构不变：电商线只改背景 / 版式 / 文案层，不改模型比例与配色。
  • 外形闸门不通过不得跳过：须回流 Prompt 模板或版式环节重试（见双引擎流与质检流）。
